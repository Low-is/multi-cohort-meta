# biomedical-literature-miner
A Python-based pipeline for automated discovery of gene expression studies from GEO using configurable biomedical search queries. Designed for RNA-seq and microarray datasets, with extensibility to broader omics data types for downstream bioinformatics analysis.

## Clone Repo
```
# Bash

git clone https://github.com/Low-is/biomedical-literature-miner.git
cd biomedical-literature-miner
```

## Create Python venv
```
# Bash

python -m venv venv
source venv/Scripts/activate # Git Bash command
```

## Install Dependencies
```
# Bash

pip install -r requirements.txt
```

## Run Literature Search Pipeline
```
# Bash

python main.py
```
